\subsection{\-Detailed \-Description}
\hyperlink{class_planner}{\-Planner} class that uses the \hyperlink{class_environment}{\-Environment} and \hyperlink{class_graph}{\-Graph} to conduct a search. 

\hyperlink{class_planner}{\-Planner} uses \hyperlink{class_graph}{\-Graph} to get successors and compute heuristics. \-It manages the priority queue for the search (sorting based on g+h) and also the closed list by using a boost hash table that maps to boolean values of whether a state is open/closed
