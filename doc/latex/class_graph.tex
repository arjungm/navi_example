\subsection{\-Detailed \-Description}
\-Intermediate class between environment and planner. 

\hyperlink{class_graph}{\-Graph} handles generating successors from a given state in the graph and it also computers the heuristic for a given state in the graph.

\-In order to extend \hyperlink{class_graph}{\-Graph} to have more nuanced motion primitves then a \-Motion\-Primitive class should be implemented, which would be used by \hyperlink{class_environment}{\-Environment} to generate real world successors. \hyperlink{class_graph}{\-Graph} would then discretize them to the appropriate \hyperlink{struct_graph_state}{\-Graph\-State}.
