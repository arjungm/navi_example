Detailed Description

Planner class that uses the Environment and Graph to conduct a search.

Planner uses Graph to get successors and compute heuristics. It manages the priority queue for the search (sorting based on g+h) and also the closed list by using a boost hash table that maps to boolean values of whether a state is open/closed
